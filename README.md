# 🛒 Amazon Clone

A responsive front-end clone of the Amazon homepage built using **HTML5** and **CSS3**. This project recreates the core layout and design of Amazon's landing page to strengthen front-end development skills, including Flexbox, responsive layouts, and modern UI styling.

---

## 🚀 Live Demo

🌐 **View the Project:**  
https://anujyad15-sudo.github.io/CloneAmazon/

---

## ✨ Features

- Amazon-inspired responsive navigation bar
- Search bar with category selector
- Delivery location section
- Sign In, Returns & Orders, and Cart sections
- Navigation panel with menu items
- Hero banner section
- Product cards displayed in a responsive grid layout
- Multi-column footer similar to Amazon
- Hover effects for interactive UI elements
- Clean and organized project structure

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- Font Awesome Icons
